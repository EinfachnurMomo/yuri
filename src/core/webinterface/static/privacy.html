YRUI COOKIE AND PRIVACY INFORMATION

Cookies

Yuri saves following information in form of localy saved cookies:

  • The uniquie ID (Snowflake) of your current session. (sessionid)
  • Your agreement of the cookie and information policy dialogue (privacyinfoaccepted)
  • Also socket.io saves cookies for identifying the current socket connection. Read more about this on their web page.

If you want to dleete all cookies saved on yuri's site, klick here (JavaScript needs to be enabled for this!).

Database

Yuri saves information in a database located on the host server of yuri's server component. All data saved in the database does only contain information generated by using the sounds boards functions. No personal information will be saved as same as your IP address, which will never be saved or displayed. All data saved in the database will be assigned to your Discord ID as key for your identity. Following data will be saved in the database:

  • The sound binding you set with the bind command of Yuri's Discord Bot, assigned to your Discord Account ID.
  • The volume number set for each guild by the volume command of Yuri's Discord Bot, assigned to the ID of the Discord server.
  • The token, you generate on the token management page with the time, when you have created the token and assigned to your Discord Account ID.

Because you have the right to delete your data from the database, please contact the hoster of the bot (which is the owner/tentant of the server this bot instance is running on, and may not be the same as the creator of this bot) and ask for delteing your data in the database by refering to your Discord ID.
